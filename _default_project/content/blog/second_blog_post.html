Duplicate this file to create a new blog post. The front matter type is blog do not change this variable.
If you don't want a image on headline, delete the front matter variables whose names start with 'feature'.
With the variable 'feature-image-left' you can influence whether the image is displayed to the left or right of the heading ( value can be 'true' or 'false' ).
